# spring-2026-is241-MO_HealthVerify
Repo for the Spring 2026 IS241.210 project team - MO HealthVerify

## Basic Structure
### Backend
This folder is where all backend code and configurations should reside. This applies to Spring Boot, Java, or even install/deployment scripts.

### Database
This folder is where all project-relevant .sql files will reside; this includes statements for creating, indexing, and linking tables, as well as CRUD queries for use in backend development.

### Frontend
This folder is where frontend website code should reside. This includes .css, .js, etc.
